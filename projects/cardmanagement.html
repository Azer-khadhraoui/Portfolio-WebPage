Card Management System

Application Web Bancaire de Gestion de Cartes - Juillet 2025

Description du Projet

Développé durant un stage chez ATB (Arab Tunisian Bank), ce système de gestion de cartes est une application web full-stack permettant la création, le suivi et la mise à jour de cartes bancaires.

Le projet utilise React.js pour l'interface utilisateur moderne et intuitive, et Node.js/Express.js pour le backend robuste. L'application est conçue pour offrir une expérience utilisateur fluide dans un environnement bancaire réel.

L'accent a été mis sur la sécurité, les performances et la livraison d'une expérience utilisateur sans faille, conformément aux standards de l'industrie bancaire.

Technologies Utilisées

React.js Node.js Express.js JavaScript HTML5 CSS3 RESTful APIs Mock Database State Management

Fonctionnalités Principales

Création de Cartes

Interface intuitive pour créer de nouvelles cartes bancaires avec validation en temps réel des données.

Suivi en Temps Réel

Tableau de bord dynamique pour suivre l'état et l'activité des cartes en temps réel.

Mise à Jour

Fonctionnalités complètes de modification et de mise à jour des informations des cartes.

Sécurité

Implémentation de mesures de sécurité robustes conformes aux standards bancaires.

APIs RESTful

Architecture backend bien structurée avec des APIs RESTful pour une communication efficace.

Gestion des Données

Système de gestion des données côté serveur avec connexion à une base de données fictive.
