Convertisseur de Devises

Application full-stack de conversion de devises avec interface moderne

Application full-stack de conversion de devises avec API REST, historique des taux et alertes personnalisées. Développée avec React, TypeScript, Node.js et une interface moderne glassmorphism.

Fonctionnalités principales

  • Conversion en temps réel entre plus de 100 devises
  • Historique des taux de change avec graphiques
  • Alertes personnalisées pour les fluctuations de devises
  • Interface responsive et moderne avec effet glassmorphism
  • API REST pour l'intégration avec d'autres applications
  • Sauvegarde des conversions favorites
  • Mise à jour automatique des taux en temps réel

Technologies utilisées

  • React avec TypeScript
  • Node.js et Express
  • API REST avec authentification
  • Design glassmorphism moderne
  • Charts.js pour les graphiques
  • Base de données pour l'historique

Architecture

  • Architecture MVC pour le backend
  • Composants React réutilisables
  • Validation TypeScript stricte
  • Optimisation des performances
