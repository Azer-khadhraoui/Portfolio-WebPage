ATB Card Management System

Application bancaire web de gestion de cartes

Contexte du stage - ATB (Arab Tunisian Bank)

Ce projet a été développé durant mon stage au siège social d'ATB (Arab Tunisian Bank) en juillet 2025. Il s'agit d'une application web full-stack de gestion de cartes bancaires, conçue pour moderniser et digitaliser les processus de gestion des cartes au sein de la banque.

Application web full-stack développée pour ATB (Arab Tunisian Bank) permettant la gestion complète des cartes bancaires. Développée avec React.js pour le frontend et Node.js/Express.js pour le backend, cette application offre une interface moderne et intuitive pour les opérations bancaires liées aux cartes.

Fonctionnalités principales

  • Système d'inscription et d'authentification sécurisé
  • Connexion utilisateur avec validation
  • Demande de nouvelle carte bancaire
  • Consultation des informations de carte
  • Tableau de bord avec menu de navigation
  • Sécurité renforcée pour les données bancaires
  • Interface responsive adaptée à tous les appareils
  • Gestion des données avec base de données fictive

Technologies utilisées

  • React.js pour l'interface utilisateur
  • Node.js et Express.js pour le backend
  • API REST pour la communication client-serveur
  • Base de données fictive pour les tests
  • CSS3 pour le design moderne
  • JavaScript ES6+
  • Design responsive

Modules de l'application

  • Inscription : Création de nouveaux comptes utilisateurs
  • Connexion : Authentification sécurisée
  • Menu principal : Navigation vers les différentes fonctionnalités
  • Demande de carte : Processus de demande de nouvelle carte
  • Consultation : Visualisation des informations de carte

Expérience acquise

  • Développement full-stack avec React et Node.js
  • Création et implémentation d'APIs REST
  • Travail en environnement bancaire professionnel
  • Compréhension des enjeux de sécurité bancaire
  • Gestion de projet en méthodologie agile
