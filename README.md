## 🚀 Installation et Utilisation

### **Méthode 1 : Accès Direct**
```bash
# Accédez directement au portfolio en ligne
https://azer-khadhraoui.github.io/Portfolio-WebPage/
```

### **Méthode 2 : Installation Locale**
```bash
# Cloner le repository
git clone https://github.com/Azer-khadhraoui/Portfolio-WebPage.git

# Naviguer dans le dossier
cd Portfolio-WebPage

# Ouvrir index.html dans votre navigateur
# Ou utiliser un serveur local (ex: Live Server)
```

### **Structure du Projet**
```
Portfolio-WebPage/
├── 📄 index.html          # Page principale
├── 📁 css/
│   └── 🎨 style.css       # Styles principaux
├── 📁 js/
```
